# 나만의 포트폴리오 웹 사이트 (KHS Portfolio)

신입 개발자 **김현수**님을 소개하는 정적 포트폴리오 웹 페이지입니다. 웹 개발 · 게임 개발 · AI 서비스 개발 학습 과정과 프로젝트를 한 곳에서 보여 줍니다.

## 무엇을 담고 있나요?

| 섹션 | 설명 |
|------|------|
| **히어로** | 인사말과 주요 활동 분야, 섹션으로 바로 이동하는 버튼 |
| **소개** | 개발 학습 방향과 관심사 |
| **경험 · 활동** | 연도별 교육(코리아 IT 아카데미, SGA 서울 게임 아카데미, 광주 인공지능사관학교) 및 학력 |
| **자격증 · 수상** | SQLD, 정보처리기사 등과 수상 실적 |
| **기술 스택** | 웹 · 게임 · AI · 도구별 태그(호버/포커스 시 학습 내용 표시) |
| **프로젝트** | 썸네일, 설명, 태그와 PDF · YouTube · GitHub 링크가 있는 카드 목록 |

접근성을 위해 **본문 건너뛰기 링크**, 키보드로 열 수 있는 **모바일 네비게이션**, **`prefers-reduced-motion`** 에 맞춘 스크롤 동작 등이 들어 있습니다.

## 기술 스택

- **HTML5** — 시맨틱 마크업, ARIA 보조
- **CSS3** — 레이아웃·타이포·반응형·다크 톤 UI (`css/style.css`)
- **JavaScript (바닐라)** — 프로젝트 카드 렌더링, 네비 토글, 연도 표시 (`js/main.js`)
- **Google Fonts** — Outfit

빌드 도구나 프레임워크 없이 브라우저에서 바로 열 수 있는 **정적 사이트**입니다.

## 폴더 구조

```
portfolio/
├── index.html          # 메인 페이지
├── css/
│   └── style.css       # 스타일
├── js/
│   └── main.js         # 프로젝트 데이터 + UI 스크립트
├── assets/
│   ├── images/         # 프로젝트 썸네일 등
│   └── docs/           # PDF 문서 등
└── README.md
```

## 로컬에서 보는 방법

1. 이 저장소를 클론하거나 폴더를 다운로드합니다.
2. `index.html`을 브라우저로 직연 하거나, 로컬 서버로 띄웁니다.

**Python이 있는 경우 (한 줄 미리보기 서버)**

```bash
# Python 3
python -m http.server 8080
```

브라우저에서 `http://localhost:8080` 으로 접속하면 됩니다.  
(파일을 `file://` 로만 열면 일부 환경에서 경로나 CORS 이슈가 날 수 있어, 가능하면 로컬 서버 사용을 권장합니다.)

## 프로젝트 카드 수정하기

카드 목록은 `js/main.js` 맨 위의 **`projects`** 배열 한 곳에서 관리합니다.

- **`title`**, **`description`**, **`tags`**: 카드에 보이는 제목·설명·기술 태그
- **`image`**: `assets/images/...` 처럼 저장소 기준 상대 경로
- **`links`**: `pdf`, `youtube`, `github` — 없는 항목은 빈 문자열 `""` 로 두면 해당 버튼이 자동으로 숨겨집니다.

항목을 배열에 추가하면 그리드에 카드가 자동으로 늘어납니다. 한글·공백이 들어간 파일명은 코드에서 경로 조각별로 인코딩해 두었습니다.

## 배포 (GitHub Pages 등)

정적 파일만 있으면 되므로 **GitHub Pages**, Netlify, Vercel 등 아무 정적 호스팅에 올려도 됩니다.  
저장소 루트에 `index.html`이 있고, `css/`, `js/`, `assets/` 구조를 그대로 유지하면 상대 경로가 깨지지 않습니다.
